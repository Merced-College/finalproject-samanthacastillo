# cpsc39-finalProjects
Samantha Castillo, Final Programming Project, CPCS-39, 12-04-2025

This is a terminal based recipe keeping app used to store recipes. It allows you to add ingredients, cooking time, directions, name and categorize each recipe. It is able to search recipes by name or catergoy and allows user input to delete any recipe by name.
